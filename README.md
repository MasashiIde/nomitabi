# ゲームライフ

## サイト概要
### サイトテーマ
気になるゲームを検索したり、おすすめのゲームを投稿したりできるSNS

### テーマを選んだ理由
ゲームをすることが好きで、様々なジャンルのゲームをするが、自分の知らないジャンルのゲームを知ることができたら便利と考え、このテーマにしました。

### ターゲットユーザ
ゲーム好きな人

### 主な利用シーン
気になるゲームを検索する時、おすすめのゲームを投稿する時
